# Todo-List
A simple and interactive Todo List application built with React. The app allows users to add new tasks, mark tasks as completed using checkboxes, and delete tasks when no longer needed. Developed using React components and state management to provide a smooth and responsive user experience while practicing core React concepts.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- User-friendly interface

## Tech Stack
- React
- JavaScript
- HTML
- CSS
